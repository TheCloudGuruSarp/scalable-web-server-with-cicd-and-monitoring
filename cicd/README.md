The continuous integration and deployment (CI/CD) pipeline for this project is configured using **GitHub Actions**.

**Workflow File Location:** `/.github/workflows/deploy.yml`

You can review the `deploy.yml` file to see the detailed steps for building, testing, and deploying the application to the AWS infrastructure.
